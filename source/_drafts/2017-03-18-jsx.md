O JSX 'e uma DSL que nos permite definir componentes no framework React misturando um codigo similar ao HTML com codigo javascript. Por exemplo, o c'doigo abaixo:

```html
<MyButton color="blue" shadowSize={2}>
  Click Me
</MyButton>
```

Ser'a convertido para:

```javascript
React.createElement(
  MyButton,
  {color: 'blue', shadowSize: 2},
  'Click Me'
)
```

Outro exemplo:

```html
<div className="sidebar" />
```

Ser'a convertido para:

```javascript
React.createElement(
  'div',
  {className: 'sidebar'},
  null
)
```

Embora seja uma nova sintaxe, no final o jsx precisar'a ser convertido para javascript para ser interpretado.
